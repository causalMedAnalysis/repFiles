# repFiles: Replication Data and Code for "Causal Mediation Analysis"

This repository contains data and code in both R and Stata for replicating the results in [Causal Mediation Analysis](https://www.cambridge.org/us/universitypress/subjects/social-science-research-methods/quantitative-methods/causal-mediation-analysis) (Wodtke and Zhou 2026, Cambridge University Press).
